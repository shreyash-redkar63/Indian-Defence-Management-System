# Indian Defence Management System

## Overview

This project is a Database Management System developed using Oracle SQL to manage defence-related information such as soldiers, ranks, units, operations, and participation records.

## Features

* Soldier Management
* Rank Management
* Unit Management
* Operation Tracking
* Participation Tracking
* Transaction Control (COMMIT, SAVEPOINT, ROLLBACK)
* Data Control (GRANT, REVOKE)
* Stored Procedures
* Functions
* Triggers
* SQL Join Queries

## Technologies Used

* Oracle SQL
* PL/SQL

## Database Tables

* Rank
* Unit
* Soldier
* Operation
* Participation

* TCL Commands
